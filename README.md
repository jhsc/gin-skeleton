## Development

### Requirements
- Install [Go](https://golang.org)
- Install [go dep](https://github.com/golang/dep)

### Makefile
```sh
// Build binary
make build

// Run linters
make lint

// Run tests
make test

// Clean build folder
make clean

// Download dependencies
make dep

// Display help menu
make help
```
